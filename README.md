este programa hecho en java sirve para registrar y controlar las ventas de una tienda por departamentos durante todo el año. Uso una matriz de 12×3, donde las filas son los meses y las columnas los departamentos: Ropa, Deportes y Juguetería.
-insertarVenta: permite ingresar o modificar una venta indicando el mes, el departamento y el monto.
-buscarVenta: busca una cantidad específica dentro de toda la matriz y muestra en qué mes y departamento se encuentra.
-eliminarVenta: elimina una venta poniendo su valor en cero según el mes y departamento elegidos.
-mostrarTabla: muestra todas las ventas en forma de tabla para que sean fáciles de leer.
-main: muestra un menú y permite usar todas las opciones hasta que el usuario decida salir.
